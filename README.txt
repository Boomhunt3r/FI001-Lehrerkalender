Lehrerkalender created by: Florian, Luis & Jannek

Please make sure the following Modules are installed:
- Flask
- Flask Login
- Flask SQLAlchemy

If not installed please install these Modules with this command line:
  On Windows:
    - py pip -m install {Module}
   On Mac/Linux:
    - pip install {Module}
  
To start the Application, open a Powershell or Termin in the folder FI001-Lehrerkalender and write in the command line:
  - py Server.py
